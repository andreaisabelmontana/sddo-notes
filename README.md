# SDDO Course Notes

Study notes for *Software Development & DevOps* — IE University · BCSAI · 2025.

Eleven topic summaries, four textbook references, three project briefs, and the
marked keys for the practice quizzes and midterm. Each topic page links to the
original course PDF.

## Structure

```
index.html              ← topic grid
topics/*.html           ← 11 topic write-ups, one per file
readings.html           ← bibliography
project.html            ← assignment briefs
quiz.html               ← past quizzes + midterm keys + exercises
pdfs/                   ← all source PDFs (content, readings, project, quiz, syllabus)
css/notes.css           ← academic palette: navy + cream + serif
```
